# Homebrew Tap for WarpDL

Official [Homebrew](https://brew.sh) tap for [WarpDL](https://github.com/warpdl/warpdl) â€” a high-performance, multi-threaded download manager.

## Installation

```bash
brew install warpdl/tap/warpdl
```

Or tap explicitly first:

```bash
brew tap warpdl/tap
brew install warpdl
```

Upgrade

```bash
brew upgrade warpdl
```

Uninstall

```bash
brew uninstall warpdl
brew untap warpdl/tap  # optional
```

Troubleshooting

If you encounter issues, try:

brew untap warpdl/tap
brew tap warpdl/tap
brew install warpdl/tap/warpdl
